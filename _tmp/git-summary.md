### Git

#### 1.1 起步 - 关于版本控制
##### 关于版本控制
版本控制时一种记录一个或若干个文件内容变化, 以便将来查阅特定版本修订情况的系统

##### 本地版本控制系统
- RCS
##### 集中化的版本控制系统 (CVCS)
- CVS
- Subversion
- Perforce
##### 分布式版本控制系统 (DVCS)
- Git
- Mercurial
- Bazaar
- Darcs

##### 问题
