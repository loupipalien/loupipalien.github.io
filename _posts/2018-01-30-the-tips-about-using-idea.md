### Alt + Enter 导入 List 没有提示 java.util.List
Settings -> Editor -> General -> Auto Import, 然后把右边页面里面的 Exclude from Import and Completion 里面的对应项移除

### Ctrl + N 查找依赖类找不到
File -> Invalidate Caches / Restart
